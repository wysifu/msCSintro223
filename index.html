computer programming


what is computer programming

computer programming, also known as coding is the process of using a programming language to give a set of instructions to a computer so it can perform a task. This is done in a step by step process.

why learn to program ?

computers are integrated in some form or fashion with almost everything in the world today, and it has been this way for some time now. computers and technology in general has helped society, and in many ways, made our life much easier.

we all know how to use these devices, but not many of us know how all of these devices work under the hood.

imagine driving from jacksonville, florida to omaha, nebraska in 1976 ? would it be easier if i gave you a device with a global positioning system capabilities ?
